## 1. 设计系统

### 1.1 CSS 自定义属性

```css
:root {
    --pri:    #667eea;        /* 主色 */
    --pri2:   #764ba2;        /* 辅助主色 */
    --grad:   linear-gradient(135deg, #667eea 0%, #764ba2 100%);  /* 主渐变 */
    --bg:     #f0f2f5;        /* 背景色 */
    --card:   #fff;           /* 卡片背景 */
    --text:   #1a1a2e;        /* 主文本 */
    --text2:  #555;           /* 次要文本 */
    --text3:  #999;           /* 辅助文本 */
    --border: #e8e8ee;        /* 边框色 */
    --radius: 14px;           /* 卡片圆角 */
    --shadow: 0 4px 24px rgba(0,0,0,0.06);           /* 默认阴影 */
    --shadow-hover: 0 8px 32px rgba(102,126,234,0.15); /* 悬浮阴影 */
}
```

### 1.2 色彩系统

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色调 | `#667eea → #764ba2` | 蓝紫渐变 |
| 成功 | `#44aa44` | 绿色 |
| 错误 | `#d63031` | 红色 |
| 错误背景 | `#fff5f5 → #ffe8e8` | 浅红渐变 |
| 输入框背景 | `#fafafc` | 浅灰 |
| 输入框聚焦 | `#fff` + `0 0 0 4px rgba(102,126,234,0.1)` | 白 + 蓝光晕 |

### 1.3 商圈标记颜色

地图和侧边栏使用 12 色循环：
```javascript
['#6366f1', '#10b981', '#f59e0b', '#ef4444',
 '#8b5cf6', '#06b6d4', '#f97316', '#ec4899',
 '#14b8a6', '#a855f7', '#3b82f6', '#84cc16']
```

### 1.4 字体

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
             "Helvetica Neue", Arial, sans-serif;
```

### 4.2 品牌标签交互

- **触发**：品牌输入框 `input` / `change` 事件实时解析
- **普通标签**：浅灰背景 `#fafafc` + 边框 + `☆` 前缀
- **必选标签**：渐变背景（主渐变） + 白色文字 + `★` 前缀
- **点击**：切换必选/普通状态
- **自动清理**：输入框中删除的品牌自动从必选列表移除
- **持久化**：`localStorage.search_required_brands` (JSON array)

### 4.3 进度区域

- 进度条：6px 高，渐变填充 + shimmer 动画
- 进度文本：13px，显示当前操作描述
- 进度详情：12px，显示品牌/页数信息
- 日志面板：可折叠，最大高度 160px，等宽字体
- 日志颜色：搜索阶段蓝色，聚类阶段紫色，生成阶段绿色

## 6. 地图渲染规范

### 6.1 标记图标 (SVG Pin)

```
尺寸：30×40px
形状：水滴形 pin（底部尖角）
填充：商圈颜色
描边：白色 2px
阴影：drop-shadow
标签：序号（白色粗体 12px）
锚点：底部中心 offset(-15, -40)
```

### 6.2 连接线

```
类型：Polyline
颜色：商圈颜色
线宽：3px
透明度：0.45
样式：dashed（虚线）
```

### 6.3 信息窗口

```
内容：门店名（商圈颜色）+ 品牌标签 + 地址
偏移：(0, -24) 像素
最小宽度：180px
```

### 6.4 地图初始化

```javascript
zoom: 13
center: 所有门店的平均坐标 [lon, lat]
setFitView(null, false, [50, 50, 50, 50])  // 四周 50px padding
```

## 8. 动画规范

| 动画名 | 用途 | 参数 |
|--------|------|------|
| `fadeUp` | 卡片/区域入场 | 0→24px 上移 + 0→1 透明度, 0.5s |
| `fadeIn` | 日志项入场 | 0→1 透明度, 0.15s |
| `slideInRight` | 右侧入场 | 0→20px 右移, ease |
| `shimmer` | 进度条闪光 | 背景位移 -200%→200%, 1.5s 循环 |
| `pulse` | 脉冲效果 | 1→0.6→1 透明度 |
| `spin` | 加载旋转 | 0→360° 旋转, 0.6s 线性循环 |
| `cardPop` | 结果卡片入场 | 16px 上移 + 0.97→1 缩放 |
| `cardIn` | 登录卡片入场 | 30px 上移 + 0.96→1 缩放, 0.6s |
| `shake` | 错误提示抖动 | ±6px 水平位移, 0.4s |
| `float` | 登录页装饰 | 30px 移动 + 缩放, 20s 循环 |

## 9. 独立 HTML 地图页

`output.py` 生成的自包含 HTML 使用不同的设计：

```
┌──────────────────────────────────────────┐
│        全屏高德地图                        │
│     (100% × 100vh)                       │
│                                          │
│                        ┌ 信息面板 ──────┐ │
│                        │ 商圈信息       │ │
│                        │ 城市: 深圳     │ │
│                        │ 找到 5 个商圈   │ │
│                        │               │ │
│                        │ ┌ 商圈 #1 ──┐ │ │
│                        │ │ ...       │ │ │
│                        │ └──────────┘ │ │
│                        └──────────────┘ │
└──────────────────────────────────────────┘
```

- 信息面板：固定右上角，300px 宽，可滚动
- 移动端：面板固定底部，100% 宽，40vh 高，顶部圆角
- 标记图标：圆形 40×40px（与 Web 内嵌版的 pin 形状不同）
- 颜色：8 色循环（`#FF4444` 系列，与 Web 端的 12 色不同）
